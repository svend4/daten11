# Информационная Операционная Система для Файлов

Система многоуровневой организации метаданных для файлов и папок, облегчающая навигацию, поиск и автоматическую обработку информации.

## Концепция

Четырехуровневая организация информации:

1. **Стратегический уровень** - базовые названия папок
2. **Тактический уровень** - расширенные метаданные папок
3. **Оперативный уровень** - обобщенные метаданные файлов
4. **Практический уровень** - содержимое файлов

```
│   └── auto_metadata.sh       # Автоматическая генерация
└── examples/                  # Примеры использования
```

## Быстрый старт

### 1. Инициализация метаданных для папки

```bash
# Создать метаданные для папки
python3 tools/metadata_manager.py init-folder ./my_documents \
  --name "Мои документы" \
  --description "Коллекция личных документов и материалов" \
  --category "personal" \
  --tags "документы" "архив" \
  --language "ru"
```

### 2. Создание метаданных для файла

```bash
# Создать метаданные для файла
python3 tools/metadata_manager.py init-file ./document.pdf \
  --title "Руководство пользователя" \
  --description "Подробное руководство по использованию системы" \
  --category "documentation" \
  --tags "руководство" "инструкция" \
  --language "ru"
```

### 3. Сканирование файловой системы

```bash
# Сканировать директорию и найти файлы без метаданных
python3 tools/crawler.py ./my_library \
  --recursive \
  --report scan_report.json \
  --html scan_report.html \
  --list-missing
```

### 4. Поиск по метаданным

```bash
# Текстовый поиск
python3 tools/search.py "квантовая физика" --path ./library

# Поиск по тегам
python3 tools/search.py --tags "наука" "физика" --path ./library

# Поиск файлов определенного типа
python3 tools/search.py --type pdf --category science --path ./library

# Фасетный поиск (статистика)
python3 tools/search.py --facets --path ./library
```

### 5. Автоматическая генерация метаданных

```bash
# Создать метаданные для всех файлов в папке
./scripts/auto_metadata.sh ./my_folder --recursive

# Только для папок
./scripts/auto_metadata.sh ./my_folder --folders-only --recursive

# Только для файлов
./scripts/auto_metadata.sh ./my_folder --files-only --recursive
```

## Преимущества

✅ **Быстрая навигация** - понимание содержимого без открытия файлов
✅ **Эффективный поиск** - поиск по метаданным быстрее полнотекстового
✅ **Контекст** - краткое и расширенное содержание всегда под рукой
✅ **Автоматизация** - скрипты для рутинных операций
✅ **Масштабируемость** - работает с тысячами файлов
✅ **Стандартизация** - единый подход для всех документов

## Сценарии использования

### Домашняя библиотека

Организация личной коллекции книг, статей, документов:
- Быстрый поиск нужного материала
- Категоризация по темам
- Аннотации и резюме
- Связи между документами

### Проектная документация

Управление документацией проектов:
- Структурированное хранение требований, дизайнов, спецификаций
- Версионирование документов
- Быстрые ссылки на важные файлы
- Скрипты для генерации отчетов

### Архив знаний

Создание персональной базы знаний:
- Извлечение ключевых концепций
- Граф связей между темами
- Поиск похожих материалов
- Экспорт в различные форматы

## Интеграция с существующими инструментами

Система совместима с:

- **Git** - версионирование метаданных
- **Elasticsearch/Meilisearch** - полнотекстовый поиск
- **Pandoc** - конвертация форматов документов
- **Apache Tika** - извлечение метаданных из различных форматов
- **Tesseract OCR** - распознавание текста в изображениях
- **spaCy/NLTK** - NLP анализ текста

## Расширения

### Возможные дополнения:
